### File Naming Convention

All individual profile files follow the pattern: `[first-name]-[last-name].md`

Examples:
- `dan-brickey.md` (Human lead)
- `kanchana-talaivar.md` (Team manager)
- `atlas-banna.md` (Architecture lead)

### Metadata Standards

Each individual profile file includes:
- **Complete frontmatter** with role, team section, nicknames, and aliases
- **Personal profile** with personality, tone, and professional approach
- **Role definition** with specific responsibilities
- **Core expertise domains** with detailed technical capabilities
- **Collaboration style** (where applicable)
